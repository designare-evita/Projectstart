KI-Integration auf Webseiten: Ein Praxisbeispiel

Wie dieses Portfolio die Gemini KI für eine interaktive Erfahrung nutzt

Veröffentlicht am 25. Juli 2024 | Lesezeit: 5 Minuten

Einleitung: Warum eine KI im Portfolio?

In der heutigen digitalen Landschaft ist es nicht mehr nur eine Spielerei, eine KI in eine Webseite zu integrieren. Es ist ein starkes Statement, das technisches Verständnis und Innovationsbereitschaft demonstriert. Dieses Portfolio nutzt Googles Gemini-Modell, um Besuchern eine direkte, interaktive und persönliche Erfahrung zu bieten – statt eines statischen Kontaktformulars können Sie direkt mit "Evita", meiner digitalen Assistentin, chatten.

Die Architektur: Sicher und Skalierbar

Die größte Herausforderung bei der Integration einer KI ist die Sicherheit. Der API-Schlüssel darf unter keinen Umständen im Frontend-Code (also im Browser des Besuchers) sichtbar sein.

  • Frontend (Was Sie sehen): Die Chatoberfläche wurde mit reinem HTML, CSS und JavaScript gebaut. Wenn Sie eine Nachricht senden, wird diese nicht direkt an Google gesendet.
  • Backend (Die sichere Brücke): Die Nachricht wird an eine Serverless Function gesendet. Das ist ein kleines, serverseitiges Skript, das auf einer Cloud-Plattform (z.B. Vercel oder Netlify) läuft.
  • Sichere API-Anfrage: Nur diese Serverless Function kennt den geheimen API-Schlüssel. Sie nimmt Ihre Frage entgegen, fügt sie in einen speziell vorbereiteten Befehl (Prompt) ein und sendet die vollständige Anfrage sicher an die Gemini-API.
  • Antwort: Die KI antwortet an die Serverless Function, welche die Antwort dann zurück an Ihr Browserfenster leitet.

Dieser Ansatz stellt sicher, dass der API-Schlüssel geschützt bleibt und die Kommunikation sicher abläuft.

Das Herzstück: Prompt-Engineering

Eine KI ist nur so gut wie die Anweisungen, die sie erhält. Dieser Prozess wird als Prompt-Engineering bezeichnet. Für "Evita" wurde ein detaillierter System-Prompt entwickelt, der ihr eine klare Persönlichkeit und genaue Anweisungen gibt. 

// Vereinfachtes Beispiel des System-Prompts in der Serverless Function
const systemPrompt = `
Du bist Evita, eine freundliche und professionelle digitale Assistentin.
Deine Aufgabe ist es, Fragen über den Inhaber dieses Portfolios zu beantworten.
Dein Wissen basiert auf diesen Informationen: [Hier werden Lebenslauf-Daten eingefügt].
Antworte immer höflich, präzise und auf Deutsch.
Lehne Anfragen ab, die nichts mit dem Portfolio oder dem Inhaber zu tun haben.
`;

Dieser Prompt stellt sicher, dass die KI im richtigen Kontext antwortet, ihre Rolle versteht und sich auf die relevanten Informationen beschränkt.

Fazit und Ausblick

Die Integration einer KI wie Gemini in eine Webseite ist mehr als nur ein technisches Gimmick. Sie zeigt die Fähigkeit, moderne, sichere und interaktive Architekturen zu entwerfen. Von der sicheren Handhabung von API-Schlüsseln über Serverless Functions bis hin zum durchdachten Prompt-Engineering demonstriert dieses Projekt ein breites Spektrum an modernen Web-Entwicklungs-Fähigkeiten.
